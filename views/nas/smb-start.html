Getting started with network shares

Hello! Are you ready to request a new network share, modify an exiting share, or perhaps setup a restricted folder?

Before we begin, you will need to know the location of the exiting share or the location for which you would like to request a new allocation, a space increase, or a modification.

The share path location will look like one of the following:

  • An NFS export path: serverName:/export/path
  • A UNC path: \\serverName\shareName
  • The path name as shown in My Computer next to the network drive letter. The network drives in My Computer will have either a UNC path like the one above or DFS path that looks something like: share$ domain\server, In either case please use the full location name.

Let's get started! Enter the full path location into the search bar below.
