Getting started with network shares

Hello! Are you ready to request a new network share, modify an exiting share, or perhaps setup a restricted folder?

Before we begin, you will need to know the network path of the share you would like work with.

If this is a new share and you want to use an existing share as a model just enter the existing share's network path into the search bar.


Enter the full network path into the search bar below.




Below are examples of what a network path will look like:

  • An NFS export path: serverName:/export/path
  • A UNC path: \\serverName\shareName
  • In My Computer, the network path is shown next to the network drive letter. The network drives will have either a UNC path like the one above or DFS path that looks something like: shareName$ domainName\serverName
